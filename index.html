My Projects

Below is a list of some of my computer science related projects. These include course projects, personal projects and bachelor final project.

Football Map Generator

Assistant Professor : Dr. Behrooz Nasihatkon
In this project, A video of a camera in a football stadium is given as input. The players and referees in the field are detected using Background subtraction and binary morphology. Each individual player or referee is then extracted from every frame and a dataset of these pictures are then fet into a convolutional neural network. Players of each team and the referees are classified.Using Sparce optical flow the players are tracked for some frames and detection is not done every frame. Then by using homography a 2D Map showing the location of players is generated as output.

Arsham On The Run (a 2D platformer game)

Assistant Professor : Dr. Mahdi Esna Ashari
Using the unity3d engine,this 2d platformer game is made like the classics. The game has custom physics and custom shaders that are made by unity shader graph. Light Weight Render Pipeline is used for performance enhancement. It also uses procedural level generation techinques to build paths intelligently in endless mode that gradually increases difficulty.

Expert System For Diagnosing Covid-19

Assistant Professor : Dr. Chitra Dadkhah
The system has three parts.First a decision tree is built with an implentation of ID3 algorithm made specific to this problem using a dataset of patients with symptoms and their diagnosis being either Covid or similar diseases like the flu.Then a knowledge base is made based on the decision tree with which the system can consult with in the last part that is the implementation of the system interface.
